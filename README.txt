Бот написан на Python
Для запуска необходимо установить следующие модули:
pyTelegramBotAPI, telebot, mysql.connector, APScheduler
Так же нужно создать базу данных и импортировать в нее файл dump.sql
После этого необходимо зайти в файл config.py и указать данные для подключения базе данных и токен от бота

Далее для запуска нужно запустить файл run.py
Если до этого все было сделано верно, то бот должен заработать.
Для работы с ботом достаточно написать ему /start и пройти простую регистрацию.
Для ознакомления с функционалом можете написать /help

С ботом можно ознакомится в рабочем состоянии:  @getInfoOdessaBot
